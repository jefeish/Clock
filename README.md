# Clock

A simple (old) Java program to demo the use of `AWT Double Buffering`.

Flicker free image rendering.

```bash
                    ***********         ***********
                    *         * ------> *         *
 [To display] <---- * Front B *   Show  * Back B. * <---- Rendering
                    *         * <------ *         *
                    ***********         ***********
```
